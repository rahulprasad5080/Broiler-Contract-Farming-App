PoultryFlow – Batch Expense & Financial Logic
Core Financial Principle
PoultryFlow maintains separate financial views for:
Company / Integrator
Farmer / Grower
Both operate independently inside the same batch.
Farmer expenses should never affect company profitability. Company expenses should never affect farmer
profitability.
This separation is mandatory throughout the app.

Each batch contains 2 expense categories:
